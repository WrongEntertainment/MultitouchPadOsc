MagicTrackpadOsc by WrongEntertainment

An Application to send multitouch messages from Magic Trackpad

MagicTrackpadOSC ist eine Applikation für das Magic Trackpad (Apple) mit dem man die eingehenden Multitouch Events als OSC-Message versenden kann. Dadurch ist es möglich,das Trackpad als Controller zu verwenden. Die MagicTrackpadOSC Applikation ist mit dem Betriebssystem MacOS 10.5+ zu benutzen, damit ein valider Support gewährleistet wird. Auswählbare OSC-Messages: trackpadname/FINGER-ID/x trackpadname/FINGER-ID/y trackpadname/FINGER-ID/xvelocity trackpadname/FINGER-ID/yveloci ty trackpadname/FINGER-ID/frame trackpadname/FINGER-ID/timestamp

Dependencies

Mac OS 10.5+

Install

Mac Java Application, no installation needed.
